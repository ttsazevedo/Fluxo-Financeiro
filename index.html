Privacidade

Seus dados ficam
com você

Arquitetura desenhada para privacidade real — não apenas no discurso.

🔒
100% Local
Todos os dados financeiros ficam armazenados apenas no seu navegador (localStorage). Nenhum servidor externo processa ou guarda suas transações.
📸
Só a imagem vai
Apenas a foto do comprovante é enviada para análise por IA. Os dados extraídos são salvos localmente — a imagem não é armazenada.
🚫
Zero anúncios
Sem rastreamento. Sem monetização de dados. Sem cookies de marketing. Você é o usuário, não o produto.
🗑️
Você controla
Limpe todos os dados a qualquer momento pelo navegador. Sem processo de exclusão complicado — os dados são seus.
Guia de Onboarding

Do zero ao controle
em 5 passos

Siga essa sequência para aproveitar ao máximo o Fluxo desde o primeiro dia.

1
⚙️ Configure categorias
Acesse Categorias e revise as 8 padrão. Ajuste orçamentos mensais conforme sua realidade.
  • Revise os orçamentos sugeridos
  • Adicione categorias específicas suas
  • Defina cores e ícones
2
💰 Registre receitas
Lance seu salário ou receitas do mês em Lançar. Isso ativa as análises de porcentagem.
  • Clique em "+ Lançamento"
  • Selecione tipo Receita
  • Informe data e valor
3
📷 Escaneie recibos
Vá em Lançar / Scan e envie fotos dos seus comprovantes. A IA faz o resto.
  • Arraste ou clique na área
  • Revise os dados extraídos
  • Confirme ou ajuste a categoria
4
📊 Analise os dados
Acesse Relatórios e explore as 3 visões. Use o filtro de período para comparar meses.
  • Competência → quando comprou
  • Caixa → quando pagou
  • Resumo → visão mensal
5
🎯 Acompanhe alertas
Na tela Alertas, veja metas sugeridas, análise de concentração e aja sobre os alertas ativos.
  • Ajuste orçamentos com base nos dados
  • Identifique categorias de maior risco
  • Use benchmarks para definir metas
